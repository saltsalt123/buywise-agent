BuyWise Agent — MVP Architecture

LangGraph Multi-Agent · Warranty / Return Decision Engine · Local-First

Workflow

  • User submits query + docs → Supervisor classifies intent
  • File Ingestion parses all source documents
  • Keyword Search retrieves relevant evidence chunks
  • Order & Policy agents extract facts
  • Verifier cross-checks → Action Agent drafts output

Tech Stack

  • Python 3.10 + LangGraph (7-node state graph)
  • FastAPI + Uvicorn (2 endpoints)
  • Pydantic V2 for state & response models
  • In-memory keyword search (BM25 fallback)
  • Multi-format parsers: PDF, CSV, EML, HTML, TXT

Eval Results

  • Evidence Recall: 0.8 (80% of gold keywords)
  • Forbidden Claims: 1.0 (no hallucinations)
  • Action Generation: 1.0 (all actions present)
  • Confidence Reported: 1.0
  • 2 test cases across warranty & return scenarios
